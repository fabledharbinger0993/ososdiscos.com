# DJ Booking Platform (Final Production Package)

Features
- Neon festival UI
- Booking system
- Event calendar
- SoundCloud + Twitch embeds
- Admin dashboard
- Drag & drop homepage builder
- Admin login system
- MongoDB database
- Docker support
- GitHub Actions auto deploy
- Seed script for demo data
- One-command start script

Quick Start

1. Configure environment
Copy `.env.example` to `.env`

2. Start platform
bash start.sh

Frontend runs on:
http://ososdiscos.com

Backend API:
